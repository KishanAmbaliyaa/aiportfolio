# Kishan Ambaliya - AI Image Generation Artist Portfolio

A modern, interactive portfolio showcasing AI image generation capabilities including:

- **Combination of Images** - Merging reference faces with product photography
- **Image Upscale** - Before/after comparison with interactive slider
- **ControlNet** - Pose-based image generation workflow
- **Image Inpainting** - Selective editing with prompts
- **Image Variation** - Theme transformations (snow, dry themes)
- **Image to Prompt** - Reverse engineering prompts from images
- **Image to Image** - Style transfer and transformations
- **Gallery** - Infinite scrolling showcase of AI-generated artwork

## Technologies Used

- HTML5, CSS3, JavaScript
- Tailwind CSS (via CDN)
- Google Fonts (DM Serif Display, Inter, Bodoni Moda, Playfair Display)
- Custom animations and interactive effects
